# **App Name**: SiteCommand

## Core Features:

- Client & Project Management: Create and organize construction projects, associating them with specific clients.
- Instruction Input: Capture client instructions efficiently via text input on site.
- Instruction Linking: Associate each recorded instruction with its respective client and project for organized data.
- Instruction Viewer: Browse and filter all recorded instructions by client or project.
- AI Instruction Summarization Tool: Automatically condense lengthy client instructions into clear, concise summaries.
- AI Action Item Extraction Tool: Identify and extract specific actionable tasks directly from the recorded instructions.

## Style Guidelines:

- Primary color: A stable, professional blue-gray (#336AB6), suggesting reliability and clarity.
- Background color: A very light, desaturated blue (#EFF4F8), providing a clean and calm canvas that enhances readability.
- Accent color: A vibrant, clear aqua (#49C3D1), used sparingly for highlights and interactive elements to draw attention.
- Headline and body text font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly legible characteristics suitable for construction site environments.
- Utilize clear, crisp, and functional outline icons to represent actions and data categories, ensuring immediate understanding on a construction site.
- Adopt a clean, structured, and responsive layout featuring card-based displays for instructions, prioritizing ease of navigation and data input on mobile devices.
- Implement subtle and purposeful animations for transitions and feedback, enhancing user experience without causing distractions.
